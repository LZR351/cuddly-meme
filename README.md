# AI求职智能匹配平台

## 项目简介

面向学生求职场景的AI智能匹配平台，帮助学生从海量岗位中快速找到与自己背景、能力、兴趣匹配度高的工作机会，并提供简历匹配度分析和针对性的优化建议，提升简历初筛通过率。

## 核心功能

### 1. 个人信息管理
- 教育背景（学历、院校、专业、毕业年份）
- 专业技能管理（标签式输入 + 熟练度评分）
- 实习/项目经验管理（支持多条记录）
- 职业兴趣选择（12个岗位方向）
- 期望薪资、求职城市

### 2. 智能岗位匹配
- **多维度加权匹配算法**：技能匹配(35%)、学历(15%)、专业(15%)、方向(15%)、经验(10%)、城市(5%)、薪资(5%)
- 内置 20+ 个真实岗位数据（覆盖前端、后端、全栈、数据分析、AI、产品、算法、设计、测试、运维、数据工程等方向）
- 支持按方向、匹配度阈值筛选
- 每个岗位展示详细的多维度匹配度分析

### 3. 简历匹配度分析
- 六维度简历评估：完整度、技能表达、经验描述、数据量化、关键词命中、结构与表达
- 自动提取简历关键信息（学历、院校、专业、技能等）
- 关键词命中率分析（对比目标岗位要求）
- 优势与不足识别

### 4. 简历优化建议
- 针对目标岗位生成个性化优化方案
- 按优先级（高/中/低）排列建议
- 提供具体的修改前后对比示例
- 自动生成优化后的简历参考模板

### 5. 可视化仪表盘
- 岗位匹配度分布饼图
- 个人技能雷达图
- 关键统计数据（匹配岗位数、平均匹配度、简历评分、优化建议数）

## 技术架构

| 模块 | 技术 |
|------|------|
| 前端框架 | 原生 HTML/CSS/JavaScript |
| 数据可视化 | Chart.js |
| 匹配算法 | 多维度加权评分（纯前端计算） |
| 简历分析 | NLP规则引擎 + 正则提取 |

## 匹配算法说明

```
综合匹配度 = 技能匹配×0.35 + 学历匹配×0.15 + 专业匹配×0.15 
           + 方向匹配×0.15 + 经验匹配×0.10 + 城市匹配×0.05 + 薪资匹配×0.05
```

- **技能匹配**：计算学生技能集与岗位需求的加权重叠度，支持技能别名和相似度计算
- **学历匹配**：对比学生学历与岗位要求，向下兼容给予基础分
- **专业匹配**：关键词匹配 + 模糊关联匹配
- **方向匹配**：直接匹配 + 相关方向映射表
- **经验匹配**：根据岗位经验要求与学生实际经历评估
- **城市匹配**：精确匹配 + 近距离城市集群

## 使用方法

1. 直接打开 `index.html` 即可运行（无需安装依赖，Chart.js 从CDN加载）
2. **第一步**：在「个人信息」页面填写教育背景、技能、经验、职业兴趣
3. **第二步**：在「智能匹配」页面点击「开始匹配」，查看推荐岗位及匹配度分析
4. **第三步**：在「简历分析」页面粘贴简历内容，获取多维度评分
5. **第四步**：在「简历优化」页面选择目标岗位，获取个性化优化建议
